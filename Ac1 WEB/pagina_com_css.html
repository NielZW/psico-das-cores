Trabalhando com estilos

CSS (Cascading Style Sheets) é o código que usamos para dar estilo às paginas HTML. Assim como o HTML o CSS não é uma linguagem de programação, é uma linguagem de folhas de estilos que permite aplicar estilos seletivamente a elementos de um documento HTML.

Como colocar estilos no HTML

Há três maneiras de colocar estilos em um documento HTML, são elas:

  • Estilos inline
  • Estilo interno
  • Estilo externo

Este é o mais recomendado para usar.
